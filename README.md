# AWS 3-Tier Student Registration System

## Project Overview

This project demonstrates a simple 3-tier web application deployed on AWS.

### Architecture

User → EC2 (Apache + PHP) → Amazon RDS (MySQL)

## AWS Services Used

- AWS EC2
- Amazon RDS
- Security Groups

## Features

- Student Registration Form
- Data Storage in MySQL Database
- Cloud Deployment on AWS

## Technologies

- HTML
- PHP
- MySQL
- AWS EC2
- Amazon RDS

## Setup

1. Launch EC2 Instance
2. Install Apache and PHP
3. Create RDS MySQL Database
4. Configure Security Groups
5. Upload Project Files
6. Update RDS Endpoint in insert.php
